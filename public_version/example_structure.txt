```
VSCode_Formatting_Pack/
│
├── .vscode/
│   ├── extensions.json                 ← Local deactivation of extensions (Prettier, RunOnSave)
│   ├── launch.json                     ← Quick run of the currently active Python script
│   ├── settings.json                   ← Full VS Code configuration (indentation, LF, UTF-8, comfort)
│   ├── keybindings.json                ← Shortcuts: Alt + M (minimap), Alt + R (reindent)
│   └── tasks.json                      ← Tasks:
│                                           - Python Formatting + Margin Cleaning (autopep8 + Margin Cleanup)
│                                           - Margin Cleaning (clean.py)
│                                           - Margin Detection (space.py)
│                                           - CRLF → LF Conversion (convert.py)
│
├── clean.py                            ← General cleanup (spaces, blank lines, quotes, normalization)
├── convert.py                          ← Convert all CRLF to LF across the project
├── space.py                            ← Detect unnecessary margins (extra spaces + blank lines)
├── backup.py                           ← Creates timestamped per-file backups on each save (no extension required)
│
├── LICENSE.txt                         ← Palks Studio Proprietary License
├── INSTALL.md                          ← Installation & usage guide
├── README_COMMERCIAL.md                ← Product presentation for Gumroad
│
└── docs/
    ├── TECHNICAL_README.md             ← Internal notes, maintenance, future versions
    └── examples/                       ← (Optional) Detailed before/after formatting examples
        ├── before.py                   ← Dirty / unstructured example files
        ├── after.py                    ← Clean, formatted versions generated by the pack
        ├── convert_lf.mp4              ← CRLF files automatically converted to LF
        ├── indent_clean.mp4            ← Broken indentation/margins fixed instantly
        ├── indent_python.mp4           ← Badly indented Python file auto-corrected
        ├── backup.mp4                  ← demonstrates automatic file backup on each save (Ctrl+S) and how to restore a deleted file from the backup folder
        └── space_clean.mp4             ← Broken file analyzed + margins auto-cleaned
```
